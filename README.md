# IoT sensors scanner

Platform, that scans network and alerts user about new found devices and their details.

## Built With

* [NMAP](https://nmap.org/) - Device Discovery/Network monitoring
* [Slack](https://slack.com/) - Used to communicate with user
* [Docker](https://www.docker.com/) - Used for maintaining and contributing project
